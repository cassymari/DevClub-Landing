## 📖 Sobre o projeto

A **DevClub Landing Page** foi desenvolvida como participação no **Concurso DevClub**.

O objetivo foi criar uma página moderna, responsiva e envolvente, capaz de transmitir a identidade do DevClub e apresentar seus conteúdos de maneira clara, dinâmica e profissional.

O projeto foi construído com **React** e organizado em componentes reutilizáveis, seções independentes e estilos separados. Também foram adicionadas animações, navegação suave, carrosséis, modais e diferentes elementos de interação.

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/cassymari/DevClub-Landing.git
```

### 2. Acesse a pasta do projeto

```bash
cd DevClub-Landing
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Inicie o servidor de desenvolvimento

```bash
npm run dev
```

Depois, abra no navegador o endereço exibido no terminal.

Normalmente:

```text
http://localhost:5173
```

---

## 📦 Gerando a versão de produção

Para gerar a versão otimizada do projeto:

```bash
npm run build
```

Os arquivos serão gerados dentro da pasta:

```text
dist
```

Para visualizar localmente a versão de produção:

```bash
npm run preview
```
